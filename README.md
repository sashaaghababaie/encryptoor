## Khamush-wrodlib
An RTL supported ```electron + reactJS``` app to view and add/edit **Khamush**-style "Entries" to/in a local JSON file.

### Install
Clone the repo and:
```
npm i
```
### Development
To run a dev server on port 3000 and open electron window in dev mode:
```
npm run dev
```

### Build
#### Mac:
```
npm run dist:mac
```
#### Windows:
```
npm run dist:win32
```
or
```
npm run dist:win64
```
