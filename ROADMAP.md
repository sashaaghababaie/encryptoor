# Roadmap

This roadmap reflects intentions, not guarantees.

---

## v1.x (Current)

- Local encrypted vault
- Password change
- Secure clipboard
- Auto-lock
- Encrypted import/export
- Automatic backups
- Auto-updates

---

## v1.x+ (Possible)

- macOS ARM64 builds
- Windows builds
- Improved import conflict resolution
- Better accessibility
- UI polish

---

## Explicitly Out of Scope

- Cloud sync
- Browser extensions
- Password recovery services
- Mobile apps

---

## Philosophy

Encryptoor will remain:

- Local-first
- Simple
- Transparent
- Opinionated toward security
